## 🛠 Стек технологій та інструменти

- **Core:** React 18 + TypeScript (для суворої типізації та надійності коду).
- **Styling:** - **SCSS (Sass):** використання препроцесора для гнучкого керування стилями (змінні, міксини, вкладеність).
  - **CSS Modules:** для повної ізоляції стилів компонентів та запобігання конфліктів імен у глобальному просторі.
  - **Container Queries (cqw):** для сучасної адаптивності елементів відносно розміру батьківського блоку.
- **Build Tool:** Vite (забезпечує миттєвий Hot Module Replacement під час розробки).
- **Deployment:** Vercel (налаштований автоматичний CI/CD деплой при кожному пуші в репозиторій).
